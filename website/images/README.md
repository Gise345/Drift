# Drift Website Images

## SVG Files Created (Convert to PNG for use)

| SVG File | Size to Export | Usage |
|----------|----------------|-------|
| `drift-og-image.svg` | 1200x630 PNG | Social sharing (Facebook, Twitter, LinkedIn) |
| `drift-logo.svg` | 200x50 PNG | Website header logo |
| `drift-logo-white.svg` | 200x50 PNG | Footer logo (dark background) |
| `favicon.svg` | 32x32 PNG | Browser tab icon |
| `apple-touch-icon.svg` | 180x180 PNG | iOS home screen bookmark |
| `social-profile.svg` | 400x400 PNG | TikTok, Instagram, Facebook profile picture |
| `google-play-badge.svg` | 646x250 PNG | Google Play download button |
| `app-store-badge.svg` | 646x250 PNG | App Store download button (Coming Soon) |

## How to Convert SVG to PNG

### Option 1: Online Tools (Easiest)
1. Go to https://svgtopng.com/ or https://cloudconvert.com/svg-to-png
2. Upload the SVG file
3. Set the dimensions listed above
4. Download the PNG

### Option 2: Figma (Free)
1. Go to https://figma.com (create free account)
2. Create new file, drag SVG onto canvas
3. Select the image, right-click > Export > PNG
4. Set dimensions and export

### Option 3: Adobe Illustrator / Photoshop
1. Open SVG in Illustrator
2. File > Export > Export As > PNG
3. Set dimensions and save
