## How to Switch Between Modes

### Switching from Rider to Driver:
1. Tap the **Profile** tab (bottom navigation)
2. Scroll all the way down
3. Tap the **purple "Switch to Driver"** button
4. You will now see the Driver Dashboard

### Switching from Driver to Rider:
1. Tap the **Menu** tab (bottom navigation)
2. Scroll all the way down
3. Tap the **purple "Switch to Rider"** button
4. You will return to the Rider home screen

### Accessing Admin Dashboard:
1. From either Rider or Driver mode, go to **Profile** tab (Rider) or **Menu** tab (Driver)
2. Scroll all the way down
3. Tap the **red "Admin Dashboard"** button
4. You will see the Admin Panel with moderation tools

## Location Permission

The app requires location access to function:
- Grant **"Allow all the time"** when prompted for full functionality
- This enables background trip tracking for safety features
- Location is used to show nearby drivers and enable navigation

## Features by Mode

**Rider Mode:**
- Request carpool rides
- View nearby drivers on map
- Add payment methods
- Share trip with contacts
- Rate drivers after trips

**Driver Mode:**
- Toggle "Go Online" to receive ride requests
- Accept or decline requests
- Navigate to pickup and destination
- View earnings and trip history

**Admin Mode:**
- Review pending driver applications
- Manage user suspensions and strikes
- View emergency alerts
- Access activity logs
